# Zacharias Zachariadis Portfolio

A clean, responsive personal portfolio website built with HTML, CSS, and JavaScript. It showcases project work, technical skills, education, and contact details in a modern one-page layout with a dedicated details page.

## Overview

This portfolio features:

- A **hero section** that welcomes visitors and links to project content.
- An **About Me** section describing academic focus and technical strengths.
- A **Projects showcase** with cards, images, and links to GitHub repos.
- A **More About Me** page for skills, languages, experience, and interests.
- A **Contact section** with email and LinkedIn links.

## Highlights

- Static site built using:
  - `index.html`
  - `details.html`
  - `Assets/main.css`
  - `Assets/script.js`
- Uses `Assets/img/` for project preview graphics.
- Includes responsive navigation and mobile-friendly layout.

## Projects Featured

- **How2Games** — gaming API platform for live game metadata.
- **DevTeam** — responsive collaborative portfolio site.
- **Fast API Generator** — Python tool for question generation from scraped web content.
- **Codecrafters Redis in Kotlin** — a simplified Redis-style key-value store.
- **Codecrafters Shell in C** — a command shell demonstrating process control and CLI design.

## How to Use

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Navigate via the top menu or the project and contact sections.
